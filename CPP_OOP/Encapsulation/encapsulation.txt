1. The data members should be labeled as private using the private access specifiers. 

2. The member function which manipulates the data members should be labeled as 
public using the public access specifier.


• Public members are accessible from any place where the class is visible

• Private members are accessible only to other members of the same class

• Protected members are accessible only to other members of the same class and to
members of classes derived from that class (Take Note).
